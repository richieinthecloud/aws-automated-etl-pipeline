# aws-automated-etl-pipeline
Using AWS services, I was able to create an automated, event-triggered data cleansing pipeline. An S3 bucket for staging raw data (bronze layer), AWS Glue for analyzing the data and schema enforcement, another S3 bucket for clean data (silver layer), Eventbridge, SNS and AWS Cloudwatch for logs.
